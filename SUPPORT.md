# Support

Where to go, by what you need:

- **Questions, ideas, "how do I…", provider-authoring help** → [GitHub Discussions](https://github.com/orgs/pluggableharness/discussions). This is the primary community channel; feature ideas start here too, and graduate to issues once they're concrete.
- **Something is broken** → [open a bug report](https://github.com/pluggableharness/agent/issues/new/choose) with the reproduction template.
- **Proposing a new provider** → the provider-proposal issue template, or a Discussion first if it's still taking shape.
- **Security vulnerabilities** → [`SECURITY.md`](SECURITY.md). Never a public issue or Discussion.

Before asking, check the docs: [`docs/specifications/`](docs/specifications/README.md) is the authoritative protocol reference (start at its README for reading order), and [`docs/first-party/`](docs/first-party/tools/README.md) catalogs the first-party tools and model providers.
